# mycoform

**真菌検査依頼フォーム｜大阪公立大学医学部附属病院 感染制御部**

近畿地区の連携医療機関向けに、病原真菌の同定・薬剤感受性検査を依頼するための構造化入力フォームです。入力データはサーバーに送信されず、ローカルPCにCSVとして保存されます。

---

## 概要

中小規模の医療施設では対応が困難な病原真菌の遺伝子解析・薬剤感受性検査を、大阪公立大学医学部附属病院 感染制御部が支援する地域連携ネットワークの一部として開発されました。

情報漏洩リスクを構造的に排除するため、以下の設計を採用しています。

- 入力データはブラウザ内で処理され、外部サーバーには一切送信されない
- CSVはローカルダウンロードのみ
- ダウンロードしたCSVを依頼施設がREDCap（院内管理）にアップロードする運用

---

## 使い方

### 1. フォームを開く

`index.html` をブラウザで開く（サーバー不要、ダブルクリックで動作）。

### 2. 依頼者情報を入力

- 依頼施設名
- 担当者名
- メールアドレス

### 3. 検体情報を入力

「＋ 検体を追加」ボタンで検体カードを追加し、以下を入力する。

| 項目 | 内容 |
|------|------|
| 検体種別 | 喀痰・血液・組織・FFPE切片など |
| 分離・採取日 | 日付 |
| 暫定同定菌種 | 依頼施設での判断（不明でも可） |
| 抗真菌薬使用歴 | あり／なし／不明 |
| 依頼内容 | 同定・感受性・FFPE検出など（複数選択可） |
| 患者背景 | 年齢層・性別・基礎疾患（匿名） |
| 備考・ファイル名 | 顕微鏡写真・コロニー写真等のファイル名 |

複数検体を一度に入力可能。

### 4. CSVをダウンロード

「CSVダウンロード」ボタンを押すと、入力内容がCSVファイルとしてローカルに保存される。ファイル名は `fungal_request_施設名_日付.csv` の形式。

### 5. REDCapにアップロード

所定のREDCapフォームに以下を入力し、CSVをアップロードする。

- 所属施設名
- 担当者名
- メールアドレス
- CSVファイル（手順4でダウンロードしたもの）

## 関連プロジェクト

本フォームは、近畿地区における医真菌症のゲノム診断体制構築と感染危機管理への応用を目的とした研究プロジェクトの一環として開発されました。収集データは菌種トレンド分析・薬剤耐性パターンの異常検知・アウトブレイク早期警告に活用される予定です。
